# workoutcomp-api
REST api for the workoutcomp project
---
## Setup
1. Setup virtual environment.
```
python3 -m venv env

//(MacOS/Linux)
source env/bin/activate
//(Windows)
env/Scripts/activate
```

2. Install dependencies
`pip install -r requirements.txt`

3. Run the server
`python manage.py runserver`
